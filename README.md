# yt-subtitle-v2

> 一个只为「我要把整个 YouTube 频道的字幕变成**我需要的语言摘要和双语字幕**」而生的极简、极快、尽量不翻车的个人神器。  
> 一条频道链接，一键到底，1000+ 视频也不怕，从此告别臃肿老项目。

## 项目简介

YouTube 自动字幕工具 v2 - 专注于频道级批量字幕处理、翻译和摘要生成。

## 核心功能

- 频道/播放列表/URL 列表批量处理
- 字幕检测（人工/自动区分）
- 增量处理（只处理新视频）
- Dry Run 模式（仅检测，不下载）
- 字幕翻译（官方优先 + AI 翻译）
- AI 摘要生成
- 统一输出目录结构
- 并发执行 + 多代理支持
- Cookie 支持
- 中英双语 UI + 4 套主题
